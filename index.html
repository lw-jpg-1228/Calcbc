📋 Unit 0: Precalculus Review

Precalculus Review — What You Need Before Calc

🔢 Algebra Essentials
• Factoring: difference of squares (a²-b² = (a-b)(a+b)), sum/difference of cubes, grouping
• Completing the square: x² + bx + (b/2)² = (x + b/2)²
• Rationalizing: multiply by conjugate to clear radicals
• Domain: all x where function is defined (watch for ÷0, √negative, log(≤0))

📐 Trigonometry
• Unit circle: Know exact values for 0, π/6, π/4, π/3, π/2 and all quadrants
• Identities you MUST know:
  sin²x + cos²x = 1
  tan²x + 1 = sec²x
  1 + cot²x = csc²x
  sin(2x) = 2sinx·cosx
  cos(2x) = cos²x - sin²x = 2cos²x - 1 = 1 - 2sin²x
  sin²x = (1 - cos2x)/2  (power-reducing, used in integration)
  cos²x = (1 + cos2x)/2

📈 Functions
• Polynomial end behavior: leading term determines behavior as x→±∞
• Rational functions: horizontal asymptotes (compare degrees), vertical asymptotes (denominator = 0)
• Exponential: y = aˣ grows/decays; y = eˣ is its own derivative (key for calc!)
• Logarithmic: y = ln(x) is inverse of eˣ; domain x > 0
• Piecewise functions: check behavior at boundary points
• Composition: (f∘g)(x) = f(g(x)) — crucial for chain rule
• Inverse functions: f(f⁻¹(x)) = x, reflect over y = x

⚡ Key Skills for Calc
• Simplify complex fractions (common denominator, then flip)
• Work with absolute value: |x| = x if x≥0, -x if x<0
• Understand function transformations (shifts, stretches, reflections)
• Convert between radians and degrees (π rad = 180°)

📋 Unit 1: Limits & Continuity (4-7%)

Unit 1: Limits & Continuity (4-7% of exam)

🎯 What is a Limit?
The value f(x) approaches as x gets close to c — even if f(c) doesn't exist there.
• lim(x→c) f(x) = L means: as x → c from BOTH sides, f(x) → L
• The limit can exist even if f(c) is undefined or different from L

📊 Finding Limits
1. Direct substitution — always try this first. If you get a number, that's the answer.
2. Indeterminate form 0/0? Try these techniques:
  • Factor and cancel: lim(x→3) (x²-9)/(x-3) = lim(x→3) (x+3) = 6
  • Rationalize: multiply by conjugate (great for √ expressions)
  • Common denominator: combine complex fractions first
3. Limits at infinity (horizontal asymptotes):
  • deg(top) < deg(bottom) → limit = 0
  • deg(top) = deg(bottom) → limit = leading coefficients ratio
  • deg(top) > deg(bottom) → limit = ±∞ (no HA)

⭐ Special Limits (MEMORIZE)
• lim(x→0) sin(x)/x = 1
• lim(x→0) (1-cos x)/x = 0
• lim(x→0) (1-cos x)/x² = 1/2
• lim(x→∞) (1+1/n)ⁿ = e
• lim(x→0) (eˣ-1)/x = 1

↔️ One-Sided Limits
• lim(x→c⁺) = limit from the RIGHT
• lim(x→c⁻) = limit from the LEFT
• Two-sided limit exists ⟺ left limit = right limit

🔗 Continuity
f is continuous at x = c if ALL THREE hold:
  1. f(c) is defined
  2. lim(x→c) f(x) exists
  3. lim(x→c) f(x) = f(c)

Types of discontinuity:
• Removable (hole): limit exists but f(c) is missing or wrong value
• Jump: left and right limits exist but aren't equal
• Infinite: function → ±∞ (vertical asymptote)

📌 Squeeze Theorem
If g(x) ≤ f(x) ≤ h(x) near c, and lim g(x) = lim h(x) = L, then lim f(x) = L
Classic example: lim(x→0) x·sin(1/x) = 0 (squeeze between -|x| and |x|)

📌 Intermediate Value Theorem (IVT)
If f is continuous on [a,b] and k is between f(a) and f(b), then ∃ c in (a,b) with f(c) = k.
• Used to guarantee a root exists: if f(a) and f(b) have opposite signs → f(c) = 0 for some c
• Requirements: f must be CONTINUOUS on the closed interval

⚠️ Common Exam Mistakes
• Forgetting that limit ≠ function value (limit can exist at a hole)
• Not checking BOTH sides for piecewise functions
• Misidentifying asymptotes: HA is about x→∞, VA is where denominator→0

📋 Unit 2: Differentiation: Definition & Properties (4-7%)

Unit 2: Differentiation — Definition & Basic Rules (4-7%)

📐 What is a Derivative?
The derivative f'(x) gives the instantaneous rate of change (slope of tangent line) at any point.
• Limit definition: f'(x) = lim(h→0) [f(x+h) - f(x)] / h
• Alternate form: f'(a) = lim(x→a) [f(x) - f(a)] / (x - a)
• Geometrically: slope of the tangent line at x = a

⚡ Differentiability
• Differentiable at a point ⟹ Continuous at that point (but NOT the reverse!)
• NOT differentiable at: corners/cusps, vertical tangents, discontinuities

📋 Exponential & Log Derivatives
• d/dx[eˣ] = eˣ
• d/dx[aˣ] = aˣ · ln(a)
• d/dx[ln x] = 1/x
• d/dx[log_a(x)] = 1/(x·ln a)

💡 Product Rule Memory Trick
(fg)' = "first times derivative of second + second times derivative of first"
Example: d/dx[x²·sin x] = x²·cos x + 2x·sin x

💡 Quotient Rule Memory Trick
"Lo d-Hi minus Hi d-Lo over Lo-Lo" = (g·f' - f·g') / g²

⚠️ Exam Tips
• Know the limit definition — it appears on FRQs
• Power rule works for negative and fractional exponents: d/dx[1/x] = d/dx[x⁻¹] = -x⁻²
• d/dx[√x] = d/dx[x^(1/2)] = 1/(2√x)

📋 Unit 3: Differentiation: Composite, Implicit, Inverse (4-7%)

Unit 3: Differentiation — Composite, Implicit, Inverse (4-7%)

⛓️ Chain Rule (THE most important rule)
d/dx[f(g(x))] = f'(g(x)) · g'(x)
"Derivative of the outside (keep inside) × derivative of the inside"

Examples:
• d/dx[sin(3x)] = cos(3x) · 3 = 3cos(3x)
• d/dx[e^(2x)] = e^(2x) · 2 = 2e^(2x)
• d/dx[(x²+1)⁵] = 5(x²+1)⁴ · 2x = 10x(x²+1)⁴
• d/dx[ln(x²)] = (1/x²) · 2x = 2/x
• d/dx[√(sin x)] = cos x / (2√(sin x))

🔄 Implicit Differentiation
When y is not isolated — differentiate both sides with respect to x, treat y as a function of x.
• Every time you differentiate a y term, multiply by dy/dx
• Then solve for dy/dx

Example: x² + y² = 25
2x + 2y·(dy/dx) = 0 → dy/dx = -x/y

🔀 Inverse Function Derivatives
If g = f⁻¹, then: g'(x) = 1 / f'(g(x))
• You need to know f' AND the value of g(x) = f⁻¹(x)
• Example: f(2) = 5, f'(2) = 3 → g'(5) = 1/f'(g(5)) = 1/f'(2) = 1/3

📋 Inverse Trig Derivatives (MEMORIZE)
FunctionDerivative

arccos x-1/√(1-x²)
arctan x1/(1+x²)
arcsec x1/(|x|√(x²-1))

📈 Higher-Order Derivatives
• f''(x) = derivative of f'(x) → gives acceleration, concavity
• Example: f(x) = e^(3x) → f'(x) = 3e^(3x) → f''(x) = 9e^(3x)

🧮 Logarithmic Differentiation
For y = xˣ or complex products: take ln of both sides first
y = xˣ → ln y = x·ln x → y'/y = ln x + 1 → y' = xˣ(ln x + 1)

⚠️ Common Mistakes
• Forgetting the chain rule (the #1 error on the AP exam!)
• Implicit: forgetting dy/dx when differentiating y terms
• Inverse trig: mixing up arcsin and arctan derivatives

📋 Unit 4: Contextual Applications of Differentiation (6-9%)

Unit 4: Contextual Applications of Differentiation (6-9%)

🚗 Motion Problems (Position/Velocity/Acceleration)
• Position: s(t) → where the particle IS
• Velocity: v(t) = s'(t) → how fast and which direction
• Acceleration: a(t) = v'(t) = s''(t) → how velocity changes
• Speed = |v(t)| (always positive)
• Speeding up: v(t) and a(t) have SAME sign
• Slowing down: v(t) and a(t) have OPPOSITE signs
• At rest: v(t) = 0
• Changes direction: v(t) changes sign

🔗 Related Rates (4-step process)
1. Draw a picture, identify all variables and rates
2. Write an equation relating the variables
3. Differentiate BOTH sides with respect to time (d/dt) — chain rule!
4. Plug in known values and solve for the unknown rate

Common setups:
• Ladder sliding: x² + y² = L² → 2x(dx/dt) + 2y(dy/dt) = 0
• Expanding circle: A = πr² → dA/dt = 2πr(dr/dt)
• Cone filling: V = (1/3)πr²h (often r and h are related)
• Shadow problems: use similar triangles

📏 Linearization (Local Linear Approximation)
L(x) = f(a) + f'(a)(x - a)
• This is the tangent line at x = a, used to approximate f(x) near a
• Example: approximate √4.1: f(x)=√x, a=4, f'(x)=1/(2√x)
  L(4.1) = 2 + (1/4)(0.1) = 2.025

🏥 L'Hôpital's Rule
If lim f(x)/g(x) gives 0/0 or ∞/∞, then:
lim f(x)/g(x) = lim f'(x)/g'(x)

• Can apply multiple times if still indeterminate
• Other forms: 0·∞ → rewrite as fraction. ∞-∞ → combine fractions. 1^∞, 0⁰, ∞⁰ → take ln first
• lim(x→∞)(1+k/x)ˣ = eᵏ (standard form)
• lim(x→0) x·ln x = 0 (rewrite as ln x/(1/x), then L'Hôp)

⚠️ Exam Traps
• Related rates: don't plug in values BEFORE differentiating
• L'Hôpital: only works for 0/0 or ∞/∞ — check the form first!
• Speed ≠ velocity (speed = |velocity|)

📋 Unit 5: Analytical Applications of Differentiation (8-11%)

Unit 5: Analytical Applications of Differentiation (8-11%)

📌 Mean Value Theorem (MVT)
If f is continuous on [a,b] and differentiable on (a,b), then:
∃ c in (a,b) such that f'(c) = [f(b) - f(a)] / (b - a)
• "There's a point where the instantaneous rate = the average rate"
• Rolle's Theorem: special case where f(a) = f(b) → f'(c) = 0

📌 Extreme Value Theorem (EVT)
If f is continuous on [a,b], then f has an absolute max AND absolute min on [a,b].

🔍 Finding Extrema
Critical points: where f'(x) = 0 or f'(x) is undefined

Candidates Test (closed interval [a,b]):
1. Find all critical points in (a,b)
2. Evaluate f at each critical point AND at endpoints a, b
3. Largest = absolute max, smallest = absolute min

First Derivative Test:
• f' changes + to − at c → local MAX
• f' changes − to + at c → local MIN
• f' doesn't change sign → neither

Second Derivative Test:
At critical point c (where f'(c) = 0):
• f''(c) < 0 → concave DOWN → local MAX 🏔️
• f''(c) > 0 → concave UP → local MIN 🏜️
• f''(c) = 0 → inconclusive (use first derivative test)

📈 Curve Sketching Checklist
f'(x)Meaning for f
f' > 0f is INCREASING ↗
f' < 0f is DECREASING ↘
f' = 0 or DNECritical point (possible max/min)
f''(x)Meaning for f
f'' > 0f is CONCAVE UP ∪ (holds water)
f'' < 0f is CONCAVE DOWN ∩
f'' = 0 and sign changeINFLECTION POINT

🎯 Optimization Problems (5-step process)
1. Draw a picture, define variables
2. Write the function to optimize
3. Write a constraint equation, use it to get one variable
4. Take derivative, set = 0, solve for critical point
5. Verify it's a max/min (second derivative test or domain argument)

⚠️ Common Mistakes
• Forgetting endpoints in candidates test
• Confusing f, f', and f'' (what each tells you)
• Inflection point: f'' must CHANGE SIGN (f''=0 alone is not enough)

📋 Unit 6: Integration & Accumulation (17-20%)

Unit 6: Integration & Accumulation of Change (17-20%) — BIGGEST unit!

🧱 Riemann Sums
Approximate ∫f(x)dx by rectangles:
• Left sum: use left endpoints → overestimates if f is DECREASING
• Right sum: use right endpoints → overestimates if f is INCREASING
• Midpoint sum: use midpoint of each subinterval
• Trapezoidal: (Δx/2)[f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)]
  → overestimates if f is CONCAVE UP

⭐ Fundamental Theorem of Calculus
FTC Part 1: d/dx[∫ₐˣ f(t) dt] = f(x)
• With chain rule: d/dx[∫ₐ^(g(x)) f(t) dt] = f(g(x)) · g'(x)
• Variable bounds on BOTH ends: split into two integrals from 0

FTC Part 2: ∫ₐᵇ f(x) dx = F(b) - F(a) where F'(x) = f(x)

🔧 u-Substitution
When you see a composite function: let u = inside, du = derivative of inside · dx
• Example: ∫x·cos(x²) dx → u = x², du = 2x dx → ½∫cos(u) du = ½sin(x²) + C
• Definite integrals: change the bounds too! Or back-substitute at the end.

🔧 [BC] Integration by Parts (IBP)
∫u dv = uv - ∫v du
• LIATE rule for choosing u: Log, Inverse trig, Algebraic, Trig, Exponential
• Pick u from left of LIATE (it gets differentiated)
• Examples: ∫x·eˣ dx (u=x, dv=eˣdx), ∫ln x dx (u=lnx, dv=dx)
• Tabular method: for ∫xⁿ·eˣ or ∫xⁿ·sinx — repeated IBP in a table
• Cycling: ∫eˣ·sinx dx — IBP twice, get original integral, solve algebraically

🔧 [BC] Partial Fractions
For rational functions P(x)/Q(x) where degree P < degree Q:
• Factor denominator completely
• Distinct linear: A/(x-a) + B/(x-b)
• Multiply through, solve for A, B by plugging in strategic x values
• Example: 1/(x²-1) = ½/(x-1) - ½/(x+1)

🔧 [BC] Improper Integrals
Type 1 — Infinite bounds: ∫₁^∞ f(x)dx = lim(t→∞) ∫₁ᵗ f(x)dx
Type 2 — Discontinuity in interval: ∫₀¹ 1/√x dx = lim(a→0⁺) ∫ₐ¹ 1/√x dx
• If the limit exists → CONVERGES. If limit = ±∞ → DIVERGES.
• Key: ∫₁^∞ 1/xᵖ dx converges iff p > 1

📐 Properties of Definite Integrals
• ∫ₐᵃ f(x)dx = 0
• ∫ₐᵇ f(x)dx = -∫ᵇₐ f(x)dx
• ∫ₐᵇ [f±g]dx = ∫ₐᵇ f dx ± ∫ₐᵇ g dx
• ∫ₐᵇ cf(x)dx = c·∫ₐᵇ f(x)dx
• ∫ₐᵇ f dx + ∫ᵇᶜ f dx = ∫ₐᶜ f dx (additivity)

📋 Unit 7: Differential Equations (6-9%)

Unit 7: Differential Equations (6-9%)

📝 What is a DE?
An equation involving a derivative: dy/dx = f(x,y)
• Solution: a function y(x) that satisfies the equation
• General solution: has a constant C
• Particular solution: uses initial condition to find C

🖼️ Slope Fields
Visual representation: draw tiny line segments with slope = dy/dx at each point
• If dy/dx depends only on x → vertical strips of equal slopes
• If dy/dx depends only on y → horizontal strips of equal slopes
• Solution curves follow the flow of the slope field

✂️ Separation of Variables (most common technique)
1. Get all y stuff on left, all x stuff on right: g(y) dy = f(x) dx
2. Integrate both sides
3. Solve for y (if possible)
4. Use initial condition to find C

Example: dy/dx = 2y, y(0) = 3
dy/y = 2dx → ln|y| = 2x + C → y = Ae^(2x) → y(0)=3 → A=3 → y = 3e^(2x)

📈 Exponential Growth/Decay
dy/dt = ky → y = Ce^(kt)
• k > 0: growth (population, compound interest)
• k < 0: decay (radioactive, cooling)
• Doubling time: t = ln(2)/k
• Half-life: t = ln(2)/|k|

📈 [BC] Logistic Growth
dy/dt = ky(1 - y/L) where L = carrying capacity
• Solution: y = L / (1 + Ce^(-kt))
• Maximum growth rate at y = L/2 (inflection point)
• As t → ∞, y → L (approaches carrying capacity)
• Starts slow, speeds up, then slows down (S-curve)

🔢 [BC] Euler's Method
Numerical approximation: y_{n+1} = yₙ + f(xₙ, yₙ) · Δx
• Start with initial point (x₀, y₀)
• Use the slope at current point to step forward
• Smaller Δx = better approximation
• Exam tip: usually 2-3 steps, show your work in a table

Example: dy/dx = x+y, y(0)=1, Δx=0.5
Step 1: y₁ = 1 + (0+1)(0.5) = 1.5 at x=0.5
Step 2: y₂ = 1.5 + (0.5+1.5)(0.5) = 2.5 at x=1.0

📋 Unit 8: Applications of Integration (6-9%)

Unit 8: Applications of Integration (6-9%)

📊 Average Value of a Function
f_avg = (1/(b-a)) · ∫ₐᵇ f(x) dx
• "The height of a rectangle with same area as under the curve"

🚗 Motion (Displacement vs Distance)
• Displacement (net change in position): ∫ₐᵇ v(t) dt — can be negative
• Total distance traveled: ∫ₐᵇ |v(t)| dt — always positive
• Net change: ∫ₐᵇ R(t) dt = total accumulated change (FTC interpretation)

📐 Area Between Curves
Horizontal slices (integrating with respect to x):
A = ∫ₐᵇ [f(x) - g(x)] dx   where f(x) ≥ g(x) on [a,b]
• "Top minus bottom"

Vertical slices (integrating with respect to y):
A = ∫ᶜᵈ [f(y) - g(y)] dy   where f(y) ≥ g(y) (right minus left)

🏛️ Volumes of Revolution
Disk method (solid, no hole): V = π∫ₐᵇ [R(x)]² dx
• R(x) = distance from axis of rotation to the curve

Washer method (with hole): V = π∫ₐᵇ ([R(x)]² - [r(x)]²) dx
• R = outer radius, r = inner radius

⚠️ Rotation about y-axis or other lines:
• About y-axis: convert to functions of y, OR use shell method
• About y = k or x = k: adjust radii (distance from curve to axis)

🔷 Volumes with Cross Sections
V = ∫ₐᵇ A(x) dx where A(x) = area of cross-section at x
• Squares: A = s² where s = distance between curves
• Equilateral triangles: A = (√3/4)s²
• Semicircles: A = (π/8)s² (diameter = s, so r = s/2)
• Isosceles right triangles: A = s²/2 (leg = s)

📏 [BC] Arc Length
Cartesian: L = ∫ₐᵇ √(1 + [f'(x)]²) dx
• Usually requires calculator (ugly integrals)
• Know the formula setup even if you can't evaluate by hand

⚠️ Common Mistakes
• Disk vs washer: draw the cross-section! If there's a gap → washer
• Rotation about a non-axis line: R ≠ f(x), R = |f(x) - axis|
• Distance vs displacement: |v(t)| for distance, v(t) for displacement

📋 Unit 9: Parametric, Polar & Vectors [BC] (11-12%)

Unit 9: Parametric, Polar & Vectors [BC Only] (11-12%)

📐 Parametric Equations
x = f(t), y = g(t) — both coordinates depend on parameter t

Parametric Derivatives:
• First derivative: dy/dx = (dy/dt) / (dx/dt)
• Second derivative: d²y/dx² = [d/dt(dy/dx)] / (dx/dt)
  ⚠️ It is NOT (d²y/dt²)/(d²x/dt²) — this is a common mistake!

Parametric Arc Length:
L = ∫ₐᵇ √[(dx/dt)² + (dy/dt)²] dt

Speed: |v(t)| = √[(x'(t))² + (y'(t))²]
Total distance: ∫ₐᵇ √[(x'(t))² + (y'(t))²] dt (same as arc length!)

🎯 Vector-Valued Functions
• Position: r⃗(t) = ⟨x(t), y(t)⟩
• Velocity: v⃗(t) = r⃗'(t) = ⟨x'(t), y'(t)⟩
• Acceleration: a⃗(t) = v⃗'(t) = ⟨x''(t), y''(t)⟩
• Speed = magnitude of velocity = |v⃗(t)|
• Displacement from t=a to t=b: ∫ₐᵇ v⃗(t) dt = ⟨∫x'(t)dt, ∫y'(t)dt⟩

🌀 Polar Coordinates
Point described by (r, θ): distance from origin and angle
• Convert to Cartesian: x = r cos θ,   y = r sin θ
• Convert to Polar: r² = x² + y²,   tan θ = y/x

Common Polar Curves:
• r = a → circle of radius a
• r = a cos θ or a sin θ → circle (diameter a)
• r = a ± b cos θ (or sin θ):
  - a = b → cardioid (heart shape)
  - a > b → limaçon without loop
  - a < b → limaçon with inner loop
• r = a cos(nθ) or a sin(nθ) → rose (n petals if n odd, 2n petals if n even)

Polar dy/dx:
dy/dx = (dy/dθ) / (dx/dθ) = (r'sin θ + r cos θ) / (r'cos θ - r sin θ)

Polar Area:
A = ½ ∫ₐᵇ [r(θ)]² dθ
• Area between two polar curves: A = ½ ∫ₐᵇ ([r₁(θ)]² - [r₂(θ)]²) dθ
• Finding bounds: set r = 0 (for one curve) or r₁ = r₂ (for intersections)

⚠️ Big Mistakes to Avoid
• d²y/dx² ≠ (d²y/dt²)/(d²x/dt²) — use the correct formula!
• Polar area uses r², not r — and don't forget the ½
• Finding all intersection points: also check where r = 0 (origin)

📋 Unit 10: Infinite Sequences & Series [BC] (17-18%)

Unit 10: Infinite Sequences & Series [BC Only] (17-18%) — BIGGEST BC unit!

📝 Sequences vs Series
• Sequence: ordered list {aₙ} = a₁, a₂, a₃, ...
• Series: sum Σaₙ = a₁ + a₂ + a₃ + ...
• A sequence converges if lim(n→∞) aₙ exists
• A series converges if its partial sums approach a finite limit

⭐ Convergence Tests (in order of usefulness)

1. nth Term Test (Divergence Test)
If lim(n→∞) aₙ ≠ 0 → series DIVERGES
⚠️ If lim = 0, this tells you NOTHING (need another test)

2. Geometric Series
Σ arⁿ: converges iff |r| < 1, sum = a/(1-r)
• First term a, common ratio r

3. p-Series
Σ 1/nᵖ: converges iff p > 1
• p = 1: harmonic series (DIVERGES!)
• p = 2: converges to π²/6

4. Integral Test
If f(x) is positive, continuous, decreasing for x ≥ 1:
Σ f(n) converges ⟺ ∫₁^∞ f(x) dx converges

5. Comparison Tests
Direct: If 0 ≤ aₙ ≤ bₙ:
  • Σbₙ converges → Σaₙ converges (smaller than convergent)
  • Σaₙ diverges → Σbₙ diverges (bigger than divergent)

Limit Comparison: If lim(aₙ/bₙ) = L where 0 < L < ∞:
  Both series do the same thing (both converge or both diverge)

6. Alternating Series Test (AST)
Σ(-1)ⁿbₙ converges if: 1) bₙ > 0, 2) bₙ is decreasing, 3) lim bₙ = 0
• AST Error Bound: |error| ≤ |first omitted term| — very useful on FRQs!

7. Ratio Test (best for factorials and exponentials)
L = lim |aₙ₊₁/aₙ|
• L < 1 → converges absolutely
• L > 1 → diverges
• L = 1 → inconclusive

🔄 Absolute vs Conditional Convergence
• Absolute: Σ|aₙ| converges → Σaₙ converges absolutely
• Conditional: Σaₙ converges but Σ|aₙ| diverges
• Example: Σ(-1)ⁿ/n converges conditionally (alt. harmonic)

📐 Power Series & Radius of Convergence
Σ cₙ(x-a)ⁿ — centered at x = a
• Use ratio test to find radius R: |x-a| < R
• Interval of convergence: check endpoints separately!
• R = 0: converges only at center. R = ∞: converges everywhere.

⭐ Taylor & Maclaurin Series
Taylor series of f(x) centered at a:
f(x) = Σ [f⁽ⁿ⁾(a)/n!] · (x-a)ⁿ
• Maclaurin = Taylor centered at a = 0

⭐ MUST MEMORIZE (6 Maclaurin Series):
FunctionSeriesInterval
eˣΣ xⁿ/n! = 1 + x + x²/2! + x³/3! + ...all x
sin xΣ (-1)ⁿx²ⁿ⁺¹/(2n+1)! = x - x³/3! + x⁵/5! - ...all x
cos xΣ (-1)ⁿx²ⁿ/(2n)! = 1 - x²/2! + x⁴/4! - ...all x
1/(1-x)Σ xⁿ = 1 + x + x² + x³ + ...|x| < 1
ln(1+x)Σ (-1)ⁿ⁺¹xⁿ/n = x - x²/2 + x³/3 - ...-1 < x ≤ 1
arctan xΣ (-1)ⁿx²ⁿ⁺¹/(2n+1) = x - x³/3 + x⁵/5 - ...|x| ≤ 1

🔧 Series Manipulation
• Substitution: e^(-x²) → replace x with -x² in eˣ series
• Differentiation: differentiate term by term
• Integration: integrate term by term
• Multiply by x: x·sin x = x·(x - x³/3! + ...) = x² - x⁴/3! + ...

📌 Lagrange Error Bound
|Rₙ(x)| ≤ M·|x-a|ⁿ⁺¹ / (n+1)!
where M = max|f⁽ⁿ⁺¹⁾(z)| for z between a and x
• For alternating series: |error| ≤ |first omitted term| (simpler!)

🗺️ Which Test to Use? (Decision flowchart)
1. Is lim aₙ ≠ 0? → nth term test: DIVERGES
2. Is it geometric (arⁿ)? → check |r|
3. Is it p-series (1/nᵖ)? → check p > 1
4. Does it alternate? → try AST
5. Has factorials or nⁿ? → try Ratio Test
6. Looks like a known series? → try Comparison
7. f(x) is easy to integrate? → try Integral Test

Y and EN

y = replaces à + place/thing → J'y vais. (I'm going there.)

en = replaces de + noun → J'en veux. (I want some.)

RuleExample

Relative Pronouns (Pronoms Relatifs)

PronounUseExample
quiSubject (who/which/that)L'homme qui parle est mon prof. (The man who speaks is my teacher.)
queDirect object (whom/which/that)Le livre que je lis est bon. (The book that I'm reading is good.)
dontOf which/whose (replaces de)La fille dont je parle est là. (The girl about whom I'm speaking is there.)
oùWhere/whenLa ville où j'habite est belle. (The city where I live is beautiful.)

⚠️ Key difference: qui is followed by a verb (it's the subject). Que is followed by a subject + verb (it's the object).

Tâche Écrite (Writing Guide) ✍️

Essay Structure

Introduction

PartWhat It DoesExample (topic: environment)
Sujet amenéGeneral opening statementAujourd'hui, l'environnement est au centre des préoccupations mondiales.
Sujet poséState the specific topic/questionLa question se pose: que pouvons-nous faire pour protéger notre planète?
Sujet diviséPreview your main pointsNous examinerons d'abord les causes de la pollution, puis les solutions possibles.

Development (Body Paragraphs)

Each paragraph follows: Topic sentence → Explanation → Example → Link to next

Aim for 2-3 body paragraphs. Each paragraph = one main idea.

Conclusion

PartWhat It DoesExample
SummaryRecap main pointsEn somme, la pollution et le gaspillage menacent notre planète.
OuvertureOpen to broader thought/questionIl reste à voir si les générations futures prendront les mesures nécessaires.
